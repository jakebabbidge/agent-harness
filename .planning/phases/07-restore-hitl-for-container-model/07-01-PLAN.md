       - Calls `query()` from `@anthropic-ai/claude-agent-sdk` with the prompt
       - Uses `canUseTool` callback to intercept `AskUserQuestion`:
         - Writes question.json to `HARNESS_IPC_DIR/question.json` (matching QuestionRecord shape)
         - Polls `HARNESS_IPC_DIR/answer.json` at 500ms interval
         - On answer, deletes answer.json and question.json, returns `{ behavior: 'deny', message: answerText }` per user decision
       - All other tools: `{ behavior: 'allow' }`
       - Iterates the async generator to completion
       - Exits with code 0 on success, 1 on error
       - Wraps everything in try/catch for error handling, cleans up question.json on unexpected exit

    2. Update `docker/Dockerfile`:
       - Add `RUN npm install -g @anthropic-ai/claude-agent-sdk` after Claude CLI install (per user decision)
       - Add `COPY agent-runner.js /usr/local/lib/agent-runner.js` (per user decision)

    3. Update `src/container/manager.ts` `createContainer`:
       - Change Cmd from `claude --dangerously-skip-permissions -p ...` to `sudo /usr/local/bin/init-firewall.sh && node /usr/local/lib/agent-runner.js`
       - Remove `promptFilePath` parameter (agent-runner reads from .harness/prompt.txt)
       - Add `Env: ['ANTHROPIC_API_KEY=' + process.env.ANTHROPIC_API_KEY, 'HARNESS_IPC_DIR=/workspace/.harness']` to container config
       - Update callers accordingly (TaskExecutor, tests)

    4. Update `src/container/manager.test.ts`:
