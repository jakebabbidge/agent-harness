<task type="auto">
  <name>Task 1: Wire CLI run and answer commands with Commander</name>
  <files>src/cli/index.ts, src/cli/run.ts, src/cli/answer.ts</files>
  <action>
    1. Create src/cli/run.ts:
       - Import: TaskExecutor from "../executor/executor.js", QuestionStore from "../hitl/question-store.js", parseWorkflow from "../workflow/parser.js", runWorkflow from "../workflow/runner.js", renderTemplate from "../template/renderer.js", { v4 as uuidv4 } from "uuid"
       - Export `async function runCommand(target: string, options: { repo?: string; variables?: string }): Promise<void>`:
         - Detect mode: if target ends with `.yaml` or `.yml` -> workflow mode; otherwise -> template mode
         - Create QuestionStore (default baseDir)
         - Create TaskExecutor(questionStore)

         **Workflow mode:**
         - Parse workflow: `const workflow = await parseWorkflow(target)`
         - Call `const result = await runWorkflow(workflow, executor)`
         - If result.status === "failed": console.error failure, process.exit(1)
         - Otherwise: console.log success, process.exit(0)

         **Template mode:**
         - Require --repo option (throw if missing)
         - Parse variables from --variables flag (JSON string, default "{}")
         - Render template: `const rendered = await renderTemplate(target, variables)`
         - Generate runId = uuidv4()
         - Execute: `const result = await executor.executeTask(rendered.rendered, options.repo, runId)`
         - console.log result summary (exitCode, resultText truncated to 200 chars)
         - process.exit(result.exitCode)

    2. Create src/cli/answer.ts:
       - Import: QuestionStore from "../hitl/question-store.js"
       - Export `async function answerCommand(runId: string, answerText: string): Promise<void>`:
         - Create QuestionStore (default baseDir)
         - Check for pending question: `const question = await questionStore.getQuestion(runId)`
         - If no question: console.error("No pending question for run {runId}"), process.exit(1)
         - Map answer to all questions: `const answers: Record<string, string> = {}; for (const q of question.questions) { answers[q.question] = answerText; }`
         - Submit: `await questionStore.submitAnswer(runId, answers)`
         - console.log("Answer submitted for run {runId}. Agent will resume shortly.")

    3. Create src/cli/index.ts:
       - Import { Command } from "commander"
       - Import { runCommand } from "./run.js"
       - Import { answerCommand } from "./answer.js"
       - Create program = new Command()
       - program.name("agent-harness").description("CLI tool for running coding agents").version("0.1.0")
       - Add `run` command:
         - `program.command("run").description("Run a template or workflow").argument("<target>", "Template file or workflow YAML").option("-r, --repo <path>", "Repository path (required for template mode)").option("-v, --variables <json>", "JSON variables for template", "{}").action(async (target, opts) => { await runCommand(target, opts); })`
       - Add `answer` command:
         - `program.command("answer").description("Answer a pending agent question").argument("<run-id>", "Run ID from the question prompt").argument("<answer>", "Answer text").action(async (runId, answer) => { await answerCommand(runId, answer); })`
       - Export program
       - program.parse(process.argv)

    4. Verify the bin entry in package.json points to dist/cli/index.js (it currently points to dist/bin/agent-harness.js — update if needed to dist/cli/index.js, OR create src/bin/agent-harness.ts that imports and runs ../cli/index.js program.parse()).

    IMPORTANT: All imports use .js extensions. Wrap action handlers in try/catch that logs error and exits with code 1. Use `process.env.ANTHROPIC_API_KEY` — if not set, print error and exit before creating executor.
  </action>
